# 🔋 dmenu-powerprofiles

power-profiles-daemon with dmenu.

![demo](/preview/demo_dmenu-powerprofiles.gif)

## 📦 Dependencies

0. `power-profiles-daemon`
1. `dmenu`
2. `jq`
3. `dunst`

## 🛠️ Installation

Just copy `dmenu-powerprofiles` into your `$PATH`.
